# facialrecognition_reactnative
App of facial recognition with React Native using camera

# Creating a new application
```
npx react-native init NameProyect
```

# Install dependences
````
npm install
````
or
```
yarn install
```

# Start in iOS
```
react-native run-ios
```

# Start in Android
```
react-native run-android
```
